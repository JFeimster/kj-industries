# K & J Industries Site Roadmap

A focused roadmap for evolving this static affiliate funding agency site without turning it into a bloated software project.

## Current Status

Completed:

- Homepage
- Team page
- Funding page
- Apply page with Tally funding intake
- Partner page with Tally partner intake
- Darwin image support
- Branded application redirect
- Premium motion layer
- Basic compliance copy

## Priority 1 — Launch Polish

- Add `og-image.jpg` social preview asset.
- Confirm final domain and update canonical URLs.
- Review every CTA on desktop and mobile.
- Compress image assets.
- Add `privacy.html` if Tally submissions will be used publicly.
- Add `terms.html` or basic disclosure page if needed.

## Priority 2 — Homepage Upgrade

- Add a stronger path-selection section:
  - Apply for Funding
  - Become a Partner
  - Explore Funding Lanes
- Add a short partner teaser.
- Add a clearer “how this works” strip.
- Add links to `funding.html`, `apply.html`, and `partner.html` above the fold.

## Priority 3 — Quality / GitHub Actions

- Add `.github/workflows/static-checks.yml`.
- Confirm required files exist.
- Confirm Tally embed URLs are present.
- Confirm stale partner mailto links are not reintroduced.
- Optionally add Proof HTML later.

## Priority 4 — Future Public Pages

Recommended pages:

```text
about.html
resources.html
faq.html
privacy.html
terms.html
thank-you.html
```

Optional pages:

```text
agents.html
broker-resources.html
funding-readiness.html
industries.html
local-business-funding.html
```

## Priority 5 — Tracking / Analytics

- Replace generic outbound links with branded `/go/*` paths.
- Add simple click tracking later if needed.
- Keep privacy and compliance implications in mind.

## Do Not Add Yet

- Next.js
- CMS
- Full CRM integration
- Heavy analytics
- Multi-step app logic
- Login / portal

Those can come later if the site proves useful. For now, keep it fast, static, and useful.
